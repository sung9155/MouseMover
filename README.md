# MouseMover

PC 절전 모드 방지 + 화면 가리기 Windows 트레이 유틸리티.

## 기능

- **절전 방지:** 주기적 마우스 지글 + Win32 `SetThreadExecutionState` API로 시스템/화면 절전 모드 방지. 지글은 사용자가 유휴 상태일 때(마지막 입력이 지글 주기 이상 경과)만 실행되어 작업 중 커서를 건드리지 않음. 절전 차단 자체는 항상 동작.
- **화면 가리기:** 모든 모니터를 검은 전체화면 오버레이로 가림.
- **덮개 중앙 시계/메시지:** 검은 덮개 중앙에 시계(디지털/아날로그 선택)와 메시지 표시. 설정에서 시계 크기·메시지 크기, 중앙 글자 색(코너와 별도), 굵게, 초 표시, 12시간제(오전/오후), 날짜 표시까지 조절.
- **절전방지 전용 모드:** 화면 덮개 없이 절전만 방지 (`절전방지만 시작`). 화면은 그대로 보이며, 자동 종료/퇴근 스케줄도 동일하게 적용. 트레이 메뉴 토글로 중지. 트레이 툴팁에 자동 해제 예정 시각이 표시됩니다 (예: `절전방지 중 (18:00까지)`).
- **상태 표시:** 오버레이 우하단에 상태 텍스트, 경과시간, 해제 안내 표시 (설정에서 각각 토글 가능).
- **트레이 활성 표시:** 절전방지(덮개 또는 절전방지 전용)가 동작 중이면 트레이 아이콘 우하단에 초록 점이 맥동해 활성 상태를 한눈에 알 수 있음. 중지하면 정적 아이콘으로 복귀.
- **즉시 해제:** 아무 키 또는 마우스 클릭으로 즉시 해제.
- **해제 예정 표기:** 자동 종료 타이머나 요일(퇴근) 스케줄이 설정돼 있으면, 해제 안내에 자동 해제 예정 시각과 남은 시간(예: `18:00 자동 해제 예정 (59분 남음)`)이 함께 표시됩니다.
- **트레이 제어:** 시스템 트레이 아이콘으로 제어 (우클릭 메뉴에서 `설정...` / `덮개 시작` / `자동 종료 ▸` / `절전방지만 시작` / `종료`). 단일 인스턴스 보장.
- **자동 종료 빠른 시작:** 트레이 우클릭 → `자동 종료 ▸ 30분/1시간/2시간/4시간`을 누르면 그 시간 뒤 자동 해제되도록 설정된 덮개가 즉시 시작 (이번 실행만 적용, 저장 설정은 변경 안 됨).

## 설정

우클릭 메뉴의 `설정...`에서 다음을 변경할 수 있습니다:

- **지글 주기:** 5–600초 (기본값: 45초)
- **상태 텍스트:** 오버레이에 표시할 텍스트 (기본값: "절전방지 중")
- **경과시간 표시:** 토글 (기본값: 켜짐)
- **해제 안내 표시:** 토글 (기본값: 켜짐)
- **덮개 중앙 시계 표시:** 토글 (기본값: 꺼짐)
- **덮개 메시지:** 중앙에 표시할 커스텀 메시지 (기본값: 비어있음)
- **시계 스타일:** 디지털/아날로그 선택 (기본값: 디지털)
- **시계 크기:** 24–200 (기본값: 64)
- **메시지 크기:** 12–120 (기본값: 24)
- **중앙 글자 색:** 색 선택 대화상자 (기본값: 회색, 코너 라벨과 별도)
- **굵게:** 토글 (기본값: 꺼짐)
- **초 표시:** 토글 (기본값: 꺼짐)
- **12시간제(오전/오후):** 토글 (기본값: 꺼짐)
- **날짜 표시:** 토글 (기본값: 꺼짐)
- **글자 크기:** 8–48pt (기본값: 11pt)
- **글자 색:** 색 선택 대화상자 (기본값: 회색)
- **Windows 시작 시 자동 실행:** 토글 (기본값: 꺼짐)
- **자동 종료:** 덮개 시작 후 일정 시간(없음/30분/1시간/2시간/4시간) 뒤 자동 해제.
- **요일 스케줄:** 근무 시간대(시작~종료)와 요일을 지정하면, 근무 시간에 켠 덮개가 근무 종료 시 자동 해제. 비근무 시간의 수동 시작은 막지 않음. (자정 넘는 근무는 미지원)
- **자동 종료 시 화면 잠금:** 토글 (기본값: 꺼짐). 켜면 덮개가 자동 종료(자동 종료 타이머 또는 퇴근 스케줄)될 때 Windows 화면을 잠급니다. 키/클릭으로 직접 해제할 때는 잠그지 않습니다. 덮개 모드에만 적용 (절전방지 전용 모드 제외).

설정은 `%APPDATA%\MouseMover\settings.json`에 자동 저장됩니다.

시각/주기 변경은 다음 "덮개 시작"부터 적용됩니다.

## 빌드

### 필수 요구사항
- .NET 9 SDK

### 명령어

```
dotnet publish src/MouseMover -c Release -r win-x64 --self-contained -p:PublishSingleFile=true
```

산출물: `src/MouseMover/bin/Release/net9.0-windows/win-x64/publish/MouseMover.exe`

## 개발

- **테스트:** `dotnet test` (28개 테스트)
- **실행:** `dotnet run --project src/MouseMover`
- **아이콘 재생성:** `powershell -ExecutionPolicy Bypass -File tools/make-icon.ps1`
